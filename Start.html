1. 首先需要一个Node.js

如果电脑安装了Node.js，请跳过该步骤



2. 打开cmd，在作业的内容根路径输入下列内容(作业名：VanillaLabs)

npm install http-server

例如我将作业拷贝在了E盘，那么cmd里就应该是：E:\VanillaLabs> npm install http-server


安装完成后，在cmd中输入以下内容：

http-server -p 8000



备注

上一步骤是为了服务JS在本地开了一个服务器。
因浏览器涉及本地文件的安全策略，一些资源文件不能以传统本地路径的方法在JS中加载。
(｡•́︿•̀｡)
在本地服务器开启的过程中，才能保证最佳的网页浏览效果。
如果需要关闭本地服务器，那么在cmd中按Ctrl + C即可


3. Enjoy!

一切准备就绪，准备漫游【香草实验室(Vanilla Labs)】。
